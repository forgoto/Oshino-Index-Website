由来 Origin

How Oshino
came to be.

Oshino Index started from a simple conversation with a friend: wouldn’t it be cool to have those Monogatari-style pause cards inside Discord?

That idea became !mono, the first command built for the bot. At the start, Oshino was mostly a personal project — a small, silly experiment made for fun. But once people started enjoying the title cards and strange little interactions, the bot slowly became something made for everyone.

From there, more systems began to form around the original idea. Coins gave users something to earn, trust gave Oshino a sense of progression, and each new command added another small piece to the world around her.

Oshino Index is still built around that same starting point: goofy, silly fun with a little bit of mystery, a little bit of chaos, and a growing collection of odd moments inside the server.

質問 FAQ

Before you begin.

What is Oshino Index designed to do?

Oshino Index adds a connected economy and progression system to Discord. Users can earn coins, collect items, build trust, draw fortunes, and interact with Oshino through a growing set of commands.

Do users need to configure anything first?

No. Most public commands are ready to use immediately. Some server-specific behaviour may depend on how the bot is set up inside the community.

What should a new user try first?

Start with !minfo or !mtuto. Then check !mprofile, complete a shift with !mwork, visit !mshop, and try an offering when you have an item available.

Where can I see recent changes?

Use !mupdates inside Discord or open the Updates page on this website for a cleaner changelog view.
